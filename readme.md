# ary

Simple terminal project setupper

## Installation

Clone the repo:

```sh
git clone https://github.com/JeffTheK/ary
```

Then run `make install` in the project directory

```sh
make install
```

## Usage

Just run ary in terminal

## Building

```sh
make install
```

### Tests

```sh
make test
```
